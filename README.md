# DJWizard
DJWizard is an AI-powered music discovery agent that takes user-supplied natural language prompts and combines them with personalized recommendations to curate playlists using audio feature analysis and Claude AI tool calling

## Stack
Frontend: React (via Vite)
Backend: Python + Flask
AI: Claude API with tool calling
Music: Spotify API
Database: Supabase
Auth: Supabase Auth
Deployment: Vercel (frontend) + Railway (backend)
